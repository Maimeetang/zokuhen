# Zokuhen (続編)

เว็บไซต์สำหรับค้นหาและดูความสัมพันธ์ภาคต่อของอนิเมะ ว่าเรื่องที่ดูมีภาคก่อน ภาคต่อ หรือเรื่องที่เกี่ยวข้องอะไรบ้าง

## Tech Stack

- Next.js · React · TypeScript
- Tailwind CSS
- React Flow — แผนภาพความสัมพันธ์
- MyAnimeList API · AniList API
